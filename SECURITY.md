# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

### How to report
- Use GitHub Private Vulnerability Reporting for this repository.
- Provide as much detail as possible to help us understand and reproduce the issue.

### Disclosure policy
- Please do not disclose vulnerabilities publicly until they have been reviewed and addressed.
- We will acknowledge receipt of your report and work to resolve confirmed issues as quickly as possible.

Thank you for helping keep OTIAdriver secure.
